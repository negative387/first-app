# Web Project

Простий веб-проєкт з HTML та CSS.

## Опис

Цей проєкт містить базові файли для веб-сайту.

## Структура проєкту

- `hello.html` - основна HTML сторінка
- `style.css` - стилі для проєкту
- `README.md` - документація проєкту

## Як використовувати

1. Відкрийте файл `hello.html` у вашому веб-браузері
2. Переглядайте вміст сторінки
